# Магия логарифмов

## О проекте

«Магия логарифмов» — обучающая 2D-платформерная игра для изучения логарифмов. Игрок решает примеры, применяет свойства логарифмов, решает уравнения и неравенства, а также использует логарифмы в научных задачах.

---

## Механика игры

### Боевая система
- Вход в триггерную зону — начало боя
- Движение игрока блокируется до решения математики
- Над врагом отображается математическое выражение
- Игрок вводит ответ в калькулятор
- После правильного ответа движение разблокируется
- Игрок добивает врага мечом

### Калькулятор
- Экранная клавиатура с цифрами, операторами и функциями
- Поддержка нижних и верхних индексов (sub/sup)
- Режимы: обычный, уравнения, неравенства
- Кнопки-маски для быстрого ввода свойств логарифмов

### Способности (маски ввода)
- Сумма логарифмов: `log_(_+_)`
- Разность логарифмов: `log_(_-_)`
- Коэффициент: `_*log_(_)`
- Степень внутри: `log_(_^_)`
